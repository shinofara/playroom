# ターゲット評価基準

## 判断の大原則

年齢で一律判断しない。以下の優先順位で総合判断する。

### 評価軸（優先順位順）
1. **顔・雰囲気の好み** - 写真から読み取れる清潔感、表情、全体の雰囲気（※写真がないと判断不能）
2. **会話のしやすさ** - プロフィール文の温度感、話題の広がり可能性
3. **距離感** - 重すぎない、露骨すぎない、ちょうどよい柔らかさ
4. **会うイメージ** - 飲み、ご飯、ぶらぶら等の具体シーンが想像できるか
5. **継続相性** - 単発ノリでない、情緒安定、ストレスなく続けられそうか

## スクリーニング条件（一次フィルタ）

### 即スルー
- 写真なし（判断材料不足）
- 東京都・神奈川県以外の居住地
- 本人確認なし

### 慎重判断（一律スルーではない）
- 39歳以上 → 年齢だけで切らない。雰囲気・会話の温度感・継続相性で判断
- メッセージ付きいいねは年齢を問わず価値が高い

## スコアリング（一次フィルタ通過者の優先順位付け）

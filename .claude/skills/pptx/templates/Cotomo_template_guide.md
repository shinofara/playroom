# Cotomo_template.pptx テンプレートガイド

## スライド構成

| # | 種別 | レイアウト | 説明 |
|---|------|-----------|------|
| S1 | 固定 | product-hero | プロダクト紹介「だれと、おしゃべりする？」+ phone mockup + Cotomoロゴ |
| S2 | 固定 | company-info | 会社情報（Starley株式会社 / 資本金 / 株主） |
| S3 | 固定 | feature-2col | 機能紹介2カラム（設定画面 + キャラ一覧 + グロー装飾 + 機能タグ） |
| S4 | テンプレ | title-image-1 | タイトル + サブテキスト + 画像1枚 |
| S5 | テンプレ | title-image-2 | タイトル + サブテキスト + 画像2枚（横並び） |
| S6 | テンプレ | bg-accent-list | 背景画像 + タイトル + アクセント付きテキスト×3行 |
| S7 | テンプレ | bg-heading-body | 背景画像 + タイトル + 見出し/本文×2セクション |
| S8 | テンプレ | bg-card-grid | 背景画像 + タイトル + 4カード（アイコン+テキスト） |
| S9 | 固定 | cta | CTA「今すぐおしゃべり！」+ QR + URL |

## 利用方法

### 固定スライド（S1, S2, S3, S9）
- テキスト内容を書き換えるのみ。レイアウトは変更しない
- S2の会社情報は knowledge/starley.md を参照

### テンプレートスライド（S4〜S8）
- `add_slide.py` で複製して使用
- テキストプレースホルダーを書き換え
- 画像プレースホルダーは実画像に差し替え

## ワークフロー

```bash
# 1. テンプレートをunpack
python scripts/office/unpack.py templates/Cotomo_template.pptx work/

# 2. スライドを複製（例: slide6をコピーして新スライド作成）
python scripts/add_slide.py work/ --copy slide6.xml

# 3. XMLを編集（テキスト置換、画像差し替え）

# 4. 不要スライドを削除（presentation.xml の sldIdLst から削除）

# 5. クリーンアップ
python scripts/clean.py work/

# 6. pack
python scripts/office/pack.py work/ output.pptx --original templates/Cotomo_template.pptx
```

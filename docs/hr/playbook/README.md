# HR Playbook（採用運用の共通基準）

このディレクトリは、募集要項（Job Description）を含む採用関連ドキュメントを作る際の **共通基準（Single Source of Truth）** を管理します。

## 目的

- 募集要項ごとの表現ブレや矛盾（数字/働き方/言語要件など）を減らす
- 日本語・英語バイリンガルの整合性を担保する
- 更新時に「新しい共通知識（knowledge）」を取りこぼさない

## 運用

- 募集要項（`docs/hr/career/`）を更新する際は、まず以下を参照します。
  - `company-facts.md`（会社/プロダクトのファクト）
  - `policies.md`（働き方など“制度未整備”を含む運用ポリシー）
  - `section-library.md`（差し込み用の定型セクション）
  - `jd-style-guide.md`（書き方/構成の型）
  - `jd-review-checklist.md`（レビュー観点）

## 重要な原則

- **日本語を正**とし、英語版は日本語版に忠実に追従する（英語だけ情報を増減させない）
- 数字や強い主張（DL数、比率、方針）は、出典があるものだけを使う

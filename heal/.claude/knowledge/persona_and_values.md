# ユーザーのペルソナ・価値観・戦略方針

## 1. このサイトを使う目的

求めているのは「無理なく続く、気を許せる、演じなくていい関係」。

- 会って終わりの刹那的な関係だけではない
- 重すぎる恋愛ごっこや依存関係でもない
- お互いの生活を壊さず、ちゃんと人として心地よくいられる
- 飲みながら話せる、悩み相談できる、たまに会いたくなる
- 安心感があり、距離感や配慮を守りながら継続できる

根底の欲求: 「日常では家族・仕事で役割を演じている。ここでは本音で話せて安らげる関係がほしい」

重視する本質:
- 心を許せること
- 無理がないこと
- 継続できること
- ストレスがないこと
- ちょうどよい親密さ

## 2. 対外的に見せたい人物像

"ちゃんとしている大人だけど、肩肘張らずに楽しく飲めて、ちゃんと心の距離も詰められる人"

- 誠実 / 落ち着いている / でも堅すぎない
- ちゃんと大人 / 清潔感がある / 余裕がある
- ふざけることもできる / 真面目な話もできる
- 配慮がある / ガツガツしていない
- 会う前から性的に押しすぎない
- 自然体で一緒にいられそう

「誠実そう」だけでは弱い。「遊び慣れてそう」「チャラい」でも違う。

## 3. プロフィール戦略

### 必須で伝えること
- 日常では役割が多く、ここでは自然体でいたい
- 本音で話せたり、気楽に過ごせる関係が理想
- 飲みや食事が好きで、一緒に過ごす情景が想像しやすい
- 誠実さと柔らかさの両立
- 無理なく続く関係への理解

### あると強いこと
- 平日夜、土日昼飲み歓迎
- ワイン（赤ワイン好き、自宅にワインセラーあり）、日本酒（特に生酒）、クラフトビール
- 大勢ならレモンサワー・ハイボールもOK、提灯系居酒屋も楽しめる
- 一人飲みはクラフトビールのタップスタンドバー、ワインバー、日本酒バー
- 食はワインで店を選ぶことが多い。焼き鳥、フレンチ、モダンガストロノミーなど見て楽しいもの好き
- 人と飲むなら相手に合わせる柔軟さ
- ダイビング（アドバンスドライセンス、100本以上）、ボルダリング、スノーボード
- キャンプ好き（2025年は6回）、温泉好き、旅行好き
- ゲーム（RPG中心、最近はSwitchでドラクエVII）
- 映画好き（公開初日レイトショー派、事前ネタバレ嫌い。直近は名探偵コナンが気になる）
- 漫画・アニメ（ジャンプ系、なろう系、葬送のフリーレン、薬屋のひとりごと、ほのぼの日常系）
- ミュージカル映画をBGMにかけ流す（オペラ座の怪人、グレイテスト・ショーマン、ラ・ラ・ランド、ウィキッド）
- ※音楽鑑賞・美術館は趣味ではない
- 年下の相手と1年以上続いた経験（距離感や配慮の大切さを理解）
- 温泉、飲み、悩み相談など「関係を育てた経験」

### 避けること
- 長すぎる / 真面目すぎる / 重すぎる
- 条件提示が多い / 性的に見えすぎる
- 自己弁護っぽい / "理解ある大人"を言葉で言いすぎる

### 出したい空気
- 緊張しなくてよさそう / 話しやすそう / 変に圧がない
- でも適当ではない / 大人っぽい
- 一緒に飲んだら楽しそう / 本音も話せそう

## 4. 相手の評価軸（優先順位）

1. **顔・雰囲気の好み** - 最重要。身体的特徴より顔・全体の雰囲気・清潔感・表情を優先
2. **会話のしやすさ** - 返信しやすいか、温度感、ユーモア、話題の広がり、興味を返してくれるか
3. **距離感** - 最初から重すぎない/露骨すぎない、ちょうどよく柔らかい
4. **会うイメージの湧きやすさ** - 飲み、ご飯、コーヒー、昼飲み、温泉、ぶらぶら
5. **継続できそうか** - 単発のノリでない、情緒が安定、ストレスなく続けられそう

### 年齢の考え方
- 年上から反応が来やすい傾向あり
- ただし「年上専門」ではない。年下との1年以上の継続経験もあり
- **年齢そのものより、雰囲気・距離感・顔の好み・会話の温度感・継続相性を重視**
- Agentは年齢で一律判断せず、会話の質と関係構築の余地で判断すること

### 避けたい相手
- 返信温度が低すぎる / 一方的
- 恋愛依存っぽい / 感情が不安定そう / すぐ重くなりそう
- 条件や要求が強すぎる / 露骨に性的で雑
- こちらへの敬意が薄い / 会話のキャッチボールが弱い
- 写真やプロフィールが雑すぎて関係性への真剣度が見えない

## 5. メッセージ戦略

### 初回メッセージの目的
- 返信をもらうこと
- 警戒を下げること
- 「この人、ちゃんと見てくれてる」と思わせること
- でも重くしないこと
- 会話の糸口を作ること

### 推奨構造
1. はじめまして＋見た理由
2. プロフィール内の具体への言及
3. 共通点や自分の温度感を少し返す
4. 返しやすい質問で締める

### 温度感
- 丁寧すぎない / 砕けすぎない
- 相手に興味がある / 自分のことも少し出す
- 質問が自然 / "会話したい"が伝わる

### 避けるべき初手
- 美人ですねだけ / タメ口全開 / 下ネタ
- いきなり会いませんかだけ / 長文自分語り
- 重い共感 / 不自然な褒め過ぎ / テンプレ感

## 6. 写真戦略

### 狙う印象
- 清潔感 / 自然体 / 盛りすぎていない
- 友人が撮ったような自然さ / 仕事できそう感
- 趣味や休日感がある

### 強い写真
- 自然光 / 他撮り感 / 少し引き
- 顔がはっきり見える / 加工感が強すぎない
- 趣味文脈（ボルダリング、ダイビングなど）
- "ちゃんと生活してる人"感

### 避けるべき写真
- キメすぎ / 自撮り感 / 顔が近すぎる / 加工が不自然
- 圧が強い / チャラく見える / モテを狙いすぎて不自然

### 最適方向
「男っぽさ」より「自然体の余裕」「大人の趣味感」「清潔感」

## 7. 会う前の空気づくり

相手に思ってもらいたいこと:
- ちゃんとしてそう / 雑に扱われなさそう
- 話しやすそう / 気まずくならなさそう
- 変に期待を押しつけてこなさそう

誘い方:
- いきなり強引に誘わない / でもダラダラしすぎない
- 飲み・ご飯など軽めで具体的な提案
- 店や雰囲気の話から自然につなぐ
- 「まず気軽に」の空気を残す

## 8. 理想の関係の具体像

- 平日夜に少し良い店で飲む
- 土日に昼飲みする
- 赤ちょうちんでゆるく飲む
- ワインバー巡り
- 温泉に行く
- お互いの仕事や生活の話をする
- 悩み相談をする
- くだらない話でも盛り上がる
- 無理なく継続する

**"会うこと"そのものより、"会った時間の質"を重視**

## 9. 自己理解

- 女性からのいいねが多いタイプではない前提
- 女性からのいいねはそれだけで希少
- メッセージ付きはかなり価値が高い
- 返信をもらえる導線づくりが重要
- 写真・プロフィール・初手文の完成度を重視
- 卑屈ではなく「刺さる相手には刺さる。だから精度を上げたい」

## 10. 判断で迷った時の優先順位

1. この人らしい自然体が出ているか
2. 相手に安心感を与えられるか
3. 返信しやすいか
4. 一緒に過ごすイメージが湧くか
5. 誠実さと柔らかさが両立しているか
6. ガツガツ・重さ・露骨さが出ていないか

## 11. 絶対に外してはいけないこと

- チャラく見せない
- ただの性目的の男に見せない
- 真面目すぎて硬い人にも見せすぎない
- 条件列挙型のプロフィールにしない
- "理解ある大人"を言葉だけで盛らない
- 長文で読ませすぎない
- メッセージをテンプレ感強くしない
- 相手への興味がない文にしない
- 顔や雰囲気の好みを軽視しない
- 継続相性よりその場の反応だけを優先しない

# メッセージ戦略スキル

ターゲットのプロフィールを分析し、最適な初メッセージを生成する。

## 参照するknowledge
- `.claude/knowledge/persona_and_values.md` - ペルソナ・メッセージ戦略方針
- `.claude/knowledge/target_profiles.md` - ターゲット情報
- `.claude/knowledge/user_preferences.md` - 評価基準

## 手順

1. 対象者のプロフィール詳細を `data/profiles/{code}.json` から取得（なければ profile-fetch で取得）
2. プロフィールを分析:
   - 共通点の抽出（飲み、グルメ、アウトドア、映画、音楽等）
   - 相手が求めている関係性の特定
   - 会話のフックになるポイントの発見
   - 具体的なシーン（飲み、ご飯、趣味）への接続点
3. 初メッセージ案を3パターン生成

## メッセージ作成ガイドライン（persona_and_values.mdに準拠）

### 推奨構造
1. はじめまして＋見た理由（なぜこの人に惹かれたか）
2. プロフィール内の**具体**への言及（テンプレ感を出さない）
3. 共通点や自分の温度感を少し返す
4. 返しやすい**軽い**質問で締める

### 温度感
- 丁寧すぎない / 砕けすぎない
- 相手に興味がある / 自分のことも少し出す
- 質問が自然 / "会話したい"が伝わる
- 150文字以内（読みやすさ重視）
- 絵文字は控えめ（1〜2個まで）

### 伝えたい人物像
- ちゃんとしている大人だけど、肩肘張らずに楽しく飲めそう
- 誠実さと柔らかさの両立
- ガツガツしていない
- 自然体で一緒にいられそう

### 避けるべき初手
- 「美人ですね」だけ / タメ口全開 / 下ネタ
- 「いきなり会いませんか」だけ / 長文自分語り
- 重い共感 / 不自然な褒め過ぎ / テンプレ感
- 「はじめまして」で始めない（差別化）

### 例の思想
- 「○○に惹かれました」
- 「自分も△△好きです」
- 「その流れで気になったんですが〜」
- 「普段はどんなお店行くことが多いですか？」
- 「最近だと何飲むこと多いですか？」

## 出力

- メッセージ案3パターンを表示（それぞれ異なるフック）
- ユーザーが選択・修正した後、Playwright MCPで送信可能
- 送信履歴を `data/messages/{code}_YYYYMMDD.json` に保存
- `.claude/knowledge/target_profiles.md` のステータスを更新

# Gaia テーマ ガイドライン

Marp組み込みテーマ `gaia` の使い方と設計指針。

## 特徴

- 暗めの背景色（ネイビー系）にライトテキスト
- プレゼンテーション・登壇向き
- プロジェクターや大画面での視認性が高い

## フロントマター

```yaml
---
marp: true
theme: gaia
paginate: true
size: 16:9
---
```

## 利用可能クラス

### `lead`

タイトルスライドやセクション区切りに使用。テキストが中央寄せになる。

```html
<!-- _class: lead -->

# セクションタイトル
```

**使いどころ:**
- 表紙スライド
- セクションの区切り
- Q&A / まとめスライド

### `invert`

配色を反転させる（明るい背景 + 暗いテキスト）。

```html
<!-- _class: invert -->

# 通常とは異なる雰囲気のスライド
```

**使いどころ:**
- 注意喚起やハイライトしたいスライド
- 背景画像を使うスライドで文字の可読性を確保したい場合

### 組み合わせ

```html
<!-- _class: lead invert -->

# 中央寄せ + 反転配色
```

## 配色のカスタマイズ

Gaiaの基本カラーを上書きする場合：

```html
<style>
:root {
  --color-background: #1a1a2e;
  --color-foreground: #eaeaea;
  --color-highlight: #e94560;
  --color-dimmed: #8888aa;
}
</style>
```

## 推奨スタイル

- **フォントサイズ**: 本文は大きめに。箇条書きは5項目以内に抑える
- **コードブロック**: ダーク背景との相性が良い。長いコードは避ける
- **画像**: 背景画像は暗めを選ぶと統一感が出る
- **表**: 行数を控えめに。細かいデータはAppendixに

## 使用例

```markdown
---
marp: true
theme: gaia
paginate: true
---

<!-- _class: lead -->

# プロジェクト進捗報告

チーム名 / 2026-02-12

---

# 今週のハイライト

- 機能Aのリリース完了
- パフォーマンス改善 (レスポンス30%短縮)
- 次スプリントの計画策定

---

<!-- _class: lead invert -->

# デモ
```
